# data-analysis-portfolio

## 👩‍💻 Overview

This repository contains academic projects completed during the "Data Storage and Analysis" course.  
The projects cover the full data lifecycle: data extraction, cleaning, transformation, analysis, visualization, and storage using relational and non-relational databases, ETL pipelines, and cloud technologies.

---

## 📊 Projects Included

### 1. Exploratory Data Analysis (EDA)
- Performed data cleaning, preprocessing, and visualization
- Identified trends, correlations, and patterns in datasets
- Created statistical summaries and business insights

### 2. Statistical Analysis – Movie Industry
- Applied statistical methods to analyze trends in the movie industry
- Performed hypothesis-based interpretation of data
- Generated insights based on distribution and relationships

### 3. Customer Churn Prediction
- Analyzed customer behavior data
- Identified factors influencing churn
- Built predictive analysis using Python

### 4. A/B Testing
- Designed and analyzed A/B experiments
- Evaluated statistical significance of results
- Supported decision-making using data-driven insights

### 5. Regression Analysis
- Built regression models to understand relationships between variables
- Evaluated model performance and interpretation of coefficients

### 6. SQL & Relational Databases
- Designed relational database schemas (Star Schema, Online Bookstore DB)
- Wrote complex SQL queries (JOINs, GROUP BY, aggregations)
- Worked with normalized database structures

### 7. ETL Pipeline (Python)
- Extracted, transformed, and loaded data using Python
- Performed data cleaning and structuring using Pandas
- Automated data processing workflows

### 8. MongoDB (NoSQL Database)
- Worked with document-based database structure
- Performed CRUD operations
- Compared relational vs non-relational database systems

### 9. Cloud Storage (AWS S3)
- Stored and managed datasets in AWS cloud
- Understood cloud-based data storage architecture
- Practiced data uploading and retrieval from S3
